ДЗ1
Відео: Основи синтаксису JS.

Відповідь:
3, 4 варианти викликають помилкую
3 варіант - тому ще нема ";" між двома виразами або просто вони написани в одну строку.
4 варіант - не закриті лапки.


ДЗ2
Відео: ключові слова LET CONST VAR

Відповідь:
1,2,3 помилкові
1 варіант - нема коми
2 варіант - не вірний запис myAge.
3 варіант - якщо такий синтаксис то має бути точне число, сума пишеться по іншому.


ДЗ3
Відео: типи данних JS

Відповідь:
1,2,3 помилкові
1 варіант - мають стояти інщі лапки - ``
2 варіант - буде значення infinity
3 варіант - буде невизначенний тип - Undefined
